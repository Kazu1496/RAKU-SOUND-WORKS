# Raku Official Website

## 技術

- Next.js（SSG/ISR）
- Vercel（Hosting/Build）
- microCMS（コンテンツ管理）
- SendGrid（メール配信）

## 開発環境

- Next.js v12.0.4
- Node.js v17.0.1
- npm v8.1.0

## 手順

1. パッケージインストール
   ```shell
   npm install
   ```
2. 環境変数を取得する
   ```shell
   npm run vercel:env:pull
   ```
3. 開発サーバー起動
   ```shell
   npm run dev
   ```
